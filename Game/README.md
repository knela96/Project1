--X-Multiply Replica--

The first lvl of the arcade game by Irem, X-Multiply

--Installation--

Just download the latest release and extecute it, there is no secret

--Controls--

W/A/S/D to move the player

SPACE to shoot missiles

There are currently three power-ups, each of them specified inside the wiki 
(the bomb, the speed up, and the tentacles(work in progress))
